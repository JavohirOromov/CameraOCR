## OCR va ML Kit haqida

### OCR (Optical Character Recognition)
**OCR** — rasm yoki suratdan matnni avtomatik aniqlash va o‘qish texnologiyasi. Kamera yoki skaner orqali olingan matnni raqamli formatga o‘tkazadi. Hujjat skanerlash, ro‘zxat yozish, matn tarjima qilish va boshqa vaziyatlarda qo‘llaniladi.

### ML Kit
**ML Kit** — Google tomonidan yaratilgan mobil qurilmalar uchun Machine Learning kutubxonasi. Oflayn ishlaydi, tez va bepul. Matn tanishi (OCR), QR-kod skanerlash, yuz aniqlash va boshqa vazifalar uchun ishlatiladi. Ilova ichida ML Kit Text Recognition modulidan foydalanib, rasmdan matn olinadi va dialog orqali ko‘rsatiladi.
